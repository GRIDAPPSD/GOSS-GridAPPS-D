Manual installation:

1 - cp opendsscmd /usr/local/bin
2 - cp *.so /usr/local/lib
3 - sudo ldconfig
4 - *.dss, *.DSS, *.sh, *.player and *.yaml are test files
5 - if you have FNCS, update with cp fncs_player /usr/local/bin
